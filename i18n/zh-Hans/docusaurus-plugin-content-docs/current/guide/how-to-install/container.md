# 如何在容器内使用 openRuyi

真的很简单！只需要拉取我们最新的 docker 镜像：

```bash
$ docker pull ghcr.io/openruyi-project/creek:latest
```

然后运行即可。欢迎使用 openRuyi！

```bash
$ docker run -it ghcr.io/openruyi-project/creek:latest
```

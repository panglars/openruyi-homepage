# openRuyi Website

Guten Tag! This is openRuyi's website code & documents repo.

## Getting Started

This project uses [Docusaurus](https://docusaurus.io).

### Installation

1. Clone this repository.
2. Change into the folder you just cloned into.
3. Run `npm install` to install the project dependencies.
4. Run `npm run` to start a live development server on `localhost:3000`.
5. Run `npm build` to generate static content into the `build` directory.
6. (Optional) Run `npm run write-translations -- --locale zh-Hans` for edit & update `zh-Hans` translations.
7. (Optional) Run `npm run start -- --locale zh-Hans` for preview `zh-Hans` translations.
